## ⚙️ Robot Design

### 🔩 Mechanical Structure
- Designed and assembled the full robot frame from scratch each season
- Custom parts modeled in SolidWorks and Fusion 360, manufactured via 3D printing
- Iterative design approach: tested, identified weaknesses, redesigned between competitions

### 🚗 Drivetrain
- Built and tuned the drivetrain for speed, maneuverability, and reliability under competition load
- Multiple iterations based on game challenge requirements each season

### 📡 Sensors & Autonomy
- Integrated encoders, color sensors, and distance sensors for autonomous operation
- Control Award recognized the quality of sensor integration and autonomous performance
